## 💡 Quy tắc chung cho tất cả các vai

1. **AI là trợ lý, Bạn là sếp**: AI sinh code/artifact, bạn là người ký duyệt (Gate Sign-off).
2. **Surgical Precision**: Dùng lệnh cụ thể (Ví dụ: _"Dùng plugin X để làm việc Y cho file Z"_) thay vì nói chung chung.
3. **Traceability**: Mọi thay đổi phải để lại dấu vết (commit message, gate summary).
