# Lessons Learned — Append-Only Log

Ghi nhận failure patterns và lessons giữa các lần chạy pipeline. Mỗi entry có format LL-NNN.

## Format

```markdown
### LL-NNN: [Title]

- **Date**: YYYY-MM-DD
- **Role**: [role phát hiện]
- **Severity**: [High/Medium/Low]
- **Trigger**: [gì triggered lesson]
- **Root cause**: [nguyên nhân gốc]
- **Fix**: [cách đã fix]
- **Prevention**: [cách ngăn recurrence]
- **ADRs affected**: [ADR IDs]
- **Quality rules affected**: [R-IDs]
```
